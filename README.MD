# ROCK PAPER SCISSORS GAME

## Introduction
This project is a simple Python program that simulates the **Rock Paper Scissors** game.  
The user plays against the computer, which randomly selects between Rock, Paper, or Scissors.
both the player and computer get to play three different hands  which is the rock, paper  and scissors 
rock destroys scissors, paper covers rock and scissors cuts paper,
so the winner depends on the hand chosen

## Features
- User-friendly text interface  
- Random computer choices  
- Displays the result of each round  
- Runs until the player chooses to stop 
- ASCII art to show the diffrent hands


---

## Requirements
- Python 3.x  
- Works on Windows, macOS, or Linux  
- Uses only the built-in `random` module  

---

## How to Run
1. Open **VS Code** or any Python IDE.  
2. Open the folder where your game file is saved (e.g. `rock_paper_scissors.py`).  
3. Open the terminal and type:
   python rock_paper_scissors.py
4. or you can also run the program by clicking on the 'run python file" BUTTON on the top right corner of your screen
